ย INTERPRETATION.
	(Amended by Ord. No. 174,861, Eff. 11/14/02, Oper. 11/1/02.)	(a)	If two or more
requirements of this Code are not identical or are in conflict, the Code
requirement that provides the greater safety to person, property or public
welfare shall prevail.	(b)	The Superintendent of Building or his duly authorized
representative is empowered to make all interpretations concerning the
provisions of this Code.
