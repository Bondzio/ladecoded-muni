                            LOS ANGELES MUNICIPAL CODE

EXISTING CONSTRUCTION. (§ 93.0109)

  EXISTING CONSTRUCTION.	Except as specified in Section 93.0111 and Subsection
(f) of Section 93.0311, nothing contained in this Code shall be construed to
curtail the uses of, nor to require any person to reinstall, reconstruct, alter,
change or remove any electric wiring or equipment which had complied with the
laws and regulations therefor in effect before the effective date of this
Code.  (Amended by Ord. No. 161,316, Eff. 7/7/86.)
