                            LOS ANGELES MUNICIPAL CODE

ALTERNATE SECURITY PROVISIONS. (§ 91.6704)

  ALTERNATE SECURITY PROVISIONS.
	(Amended by Ord. No. 171,175, Eff. 7/25/96.)	The provisions of this division
are not intended to prevent the use of any device, hardware, or method of
construction, not specifically prescribed in this division, when such alternate
provides equivalent security and is approved by the Department.
