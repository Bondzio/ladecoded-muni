                            LOS ANGELES MUNICIPAL CODE

PERMIT REQUIRED. (§ 57.14.03)

  PERMIT REQUIRED.
	(Added by Ord. No. 172,043, Eff. 7/13/98.)	No person shall operate or maintain
a new or existing Unified Program Facility without having obtained an annually
renewable Unified Program Facility Permit  with the appropriate authorization
for each applicable unified program element pursuant to this division, or other
authorized Permit.
