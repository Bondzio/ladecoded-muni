                            LOS ANGELES MUNICIPAL CODE

DEFINITIONS. (§ 91.5.202)

  DEFINITIONS.	Section 202 of the CRC is adopted by reference, except that the
following CRC definitions are not adopted:	ATTIC, HABITABLE.	BUILDING
OFFICIAL.	FAMILY.	LOT.	The following definitions are adopted:	BUILDING
OFFICIAL.  See Los Angeles Municipal Code Section 91.202.	LOT.  See Los
Angeles Municipal Code Section 12.03.	SUPERINTENDENT OF BUILDING.  See Los
Angeles Municipal Code Section 91.202.
