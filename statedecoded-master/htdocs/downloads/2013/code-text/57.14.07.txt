                            LOS ANGELES MUNICIPAL CODE

PERMITS – POSTING AND KEEPING. (§ 57.14.07)

  PERMITS – POSTING AND KEEPING.
	(Added by Ord. No. 172,043, Eff. 7/13/98.)	Posting and Keeping.	Each Permit
issued pursuant to the provisions of this division shall be posted in a
conspicuous place on the premises for which the same is issued.
