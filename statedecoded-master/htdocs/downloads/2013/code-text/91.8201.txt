                            LOS ANGELES MUNICIPAL CODE

GENERAL. (§ 91.8201)

  GENERAL.
	(Amended by Ord. No. 179,324, Eff. 12/10/07, Oper. 1/1/08.)	Every change of
occupancy, use and rating classification in any existing building or structure
shall conform to the construction requirements for the group occupancy to be
housed in the building or structure or for the use to which the building or
structure is to be put, as set forth in this division.
